# VøR — Noctyra Monster Deck

**Project:** Noctyra Monster Deck  
**Owner:** Timothy Wesley Stone  
**Protocol:** Strong Mint v2

## Name Usage

**Noctyra** (full name):
- Story, lore, world-building
- Formal documentation
- First-time explanations
- URL path: `/noctyra/`

**VøR** (shorthand):
- Community shorthand
- Technical contexts
- Protocol strings (ASCII: `vor`, UTF-8: `vør`)
- Internal references

## Pronunciation

- **Noctyra:** /nok-TIE-ruh/ or /NOK-tir-uh/
- **VøR:** Undefined (intentionally ambiguous)

## Structure

```
noctyra/
├── cards/              # Monster card .md files
├── assets/             # Styling and scripts
│   ├── css/

├── index.html          # Card display page
```

## Design Intent

The world has a long name for narrative weight. People will use the short name because they always do. Both are intentional. Both are correct.
